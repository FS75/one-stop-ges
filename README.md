# One Stop GES

An all-encompassing dashboard providing comprehensive Graduate Employment Survey (GES) results from various universities in Singapore, accumulated over the years.

<!-- ## Link: [One Stop GES]() -->

## Installation

1. Install all required dependencies
```
npm install
```
2. Run the app
```
npm run
```
